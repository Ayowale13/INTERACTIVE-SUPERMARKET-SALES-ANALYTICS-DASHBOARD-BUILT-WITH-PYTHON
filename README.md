# Supermarket Sales Analytics Dashboard

##  Project Overview
This project is a complete interactive dashboard application built from scratch using Python and Tkinter. It transforms raw supermarket sales data into actionable insights through a polished GUI that combines frontend design, backend logic, and data analytics.

##  Features

### Interactive Interface
- Multi-page navigation with smooth transitions  
- Buttons triggering real-time data analysis  
- User-friendly forms and UI components  
- Clean greenâ€“white design for clarity and simplicity  

### Backend Logic & Data Handling
- Automated sales calculations  
- Data processing with pandas  
- Dynamic filtering and grouping of data  
- Robust error handling with responsive UI feedback  

### Visual Analytics & Insights
- Customer demographics breakdown (male vs female, customer types)  
- Branch performance comparisons  
- Product line sales analysis  
- Unit price distribution  
- Purchase quantity trends  
- Product category insights with actionable recommendations  
- Embedded charts integrated directly into the Tkinter interface
